# Code of Conduct

This project is small, calm, and friendly. We want the community around it to feel the same way the
game does: welcoming, forgiving, and low-stress.

## The Short Version

Be kind. Assume good faith. Help people glide.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless
of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity
and expression, level of experience, education, socio-economic status, nationality, personal
appearance, race, religion, or sexual identity and orientation.

## What We Encourage

- Patience with newcomers and first-time contributors.
- Clear, respectful, and constructive feedback.
- Accessibility-minded thinking — this project treats inclusion as a feature.
- Crediting others' work and ideas.
- Gracefully accepting that not every change will be merged.

## What Is Not Okay

- Harassment, insults, or derogatory comments, public or private.
- Discriminatory or demeaning language or imagery.
- Personal or political attacks.
- Publishing others' private information without explicit permission.
- Sustained disruption of discussions, issues, or pull requests.

## Scope

This Code of Conduct applies in all project spaces — issues, pull requests, commits, and discussions —
and whenever an individual is representing the project in public spaces.

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the project
maintainer through a [GitHub private security advisory](https://github.com/0thernes/glidiebirdie/security/advisories/new)
or by opening a minimal issue asking for a private channel without posting sensitive details. All
reports will be reviewed and handled with discretion.

## Enforcement

Maintainers may edit, hide, or remove contributions that violate this Code of Conduct, and may
temporarily or permanently restrict participation for behavior they deem inappropriate, threatening,
or harmful.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1, trimmed and tuned to fit a small, calm, solo-built project.
